Cloudflare Pages 很适合静态博客：仓库推送后自动构建，构建产物直接分发到边缘节点。

## 构建命令

```bash
npm run build
```

## 输出目录

```bash
dist
```

## 注意事项

上线前记得修改 `astro.config.mjs` 里的 `site` 字段。RSS、Sitemap 和社交分享都会依赖这个域名。
